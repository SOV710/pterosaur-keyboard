# PCB 方案说明和替代方案

## 本项目的布线方案

**重要说明：本项目是纯手工布线（Hand-wired）方案，没有 PCB 设计文件。**

### 手工布线的特点

#### 优点：

1. **完全定制：** 可以随意修改键盘布局而无需重新设计 PCB
2. **成本较低：** 不需要订购 PCB（但需要更多人工时间）
3. **易于调试：** 所有连接可见，方便排查问题
4. **可维护性：** 使用排针连接，所有部件可拆卸更换

#### 缺点：

1. **耗时：** 焊接和布线需要大量时间（每侧 8-12 小时）
2. **美观度：** 内部布线可能不如 PCB 整洁
3. **可靠性：** 细线容易断裂（README 中特别提到）
4. **技能要求：** 需要一定的焊接经验

## 手工布线详细方案

### 原理图

键盘使用标准的矩阵扫描方案：

```
         Col0    Col1    Col2    Col3    Col4
        ┌───┐   ┌───┐   ┌───┐   ┌───┐   ┌───┐
Row0 ───┤ ▶ ├───┤ ▶ ├───┤ ▶ ├───┤ ▶ ├───┤ ▶ ├───
        └───┘   └───┘   └───┘   └───┘   └───┘
         │       │       │       │       │
        ┌───┐   ┌───┐   ┌───┐   ┌───┐   ┌───┐
Row1 ───┤ ▶ ├───┤ ▶ ├───┤ ▶ ├───┤ ▶ ├───┤ ▶ ├───
        └───┘   └───┘   └───┘   └───┘   └───┘
         │       │       │       │       │
        ┌───┐   ┌───┐   ┌───┐   ┌───┐   ┌───┐
Row2 ───┤ ▶ ├───┤ ▶ ├───┤ ▶ ├───┤ ▶ ├───┤ ▶ ├───
        └───┘   └───┘   └───┘   └───┘   └───┘
         │       │       │       │       │
         ↓       ↓       ↓       ↓       ↓
        to Pro Micro Columns
```

- **▶** = 二极管（Diode）
- **横线** = 行（Row）连接
- **竖线** = 列（Column）连接

### 3x5 布局的布线

#### 需要的连接：

1. **行连接：** 3 根（每行一根导线连接所有二极管）
2. **列连接：** 5 根（每列一根导线连接所有开关）
3. **Pro Micro 引脚：** 3 + 5 = 8 个 GPIO
4. **TRRS 通信：** 1 个引脚（PD0）+ VCC + GND
5. **轨迹球（可选）：** 7 个引脚（VCC, GND, MISO, MOSI, SCK, NCS, Motion）

#### 建议的引脚分配（3x5 右侧主控）：

```
Pro Micro 引脚分配：

行（Rows）：
- Row 0: D4
- Row 1: D5
- Row 2: D6

列（Columns）：
- Col 0: A0 (F7)
- Col 1: A1 (F6)
- Col 2: A2 (F5)
- Col 3: A3 (F4)
- Col 4: F4 (或其他可用引脚如 B5)

TRRS 通信：
- PD0 (Serial Data)
- VCC
- GND

轨迹球（PMW3360）：
- VCC: VCC
- GND: GND
- MISO: B4
- MOSI: B2
- SCK: B1
- NCS: B0
- Motion: 可选（用于检测运动）
```

**注意：** 左侧使用相同的引脚布局。

### 布线材料

| 材料 | 规格 | 用途 | 数量（3x5 双手） |
|------|------|------|------------------|
| 彩色导线 | 28-30 AWG | 行列连接 | 约 5-10 米 |
| 二极管 | 1N4148 | 每个开关一个 | 38+ |
| 排针 | 2.54mm | 可拆卸连接 | 若干排 |
| 排母 | 2.54mm | 可拆卸连接 | 若干排 |
| 热缩管 | 2-3mm | 绝缘保护 | 1 卷 |
| Kailh 热插拔座 | Choc 专用 | 开关固定 | 38+ |

### 布线技巧

1. **颜色编码：**
   - 使用不同颜色区分行和列
   - 例如：行用暖色（红、橙、黄），列用冷色（蓝、绿、紫）

2. **预留长度：**
   - 外壳外焊接时，预留足够长度以便装入外壳
   - 拇指区到主区域的连线要特别留足

3. **使用排针：**
   - 行和列的末端连接排针
   - Pro Micro 使用排母，方便更换
   - 轨迹球传感器也使用排针/排母

4. **绝缘处理：**
   - 所有裸露焊点用热缩管包裹
   - 或使用绝缘胶带

5. **应力释放：**
   - 在线缆可能受拉扯的地方用热熔胶固定
   - Pro Micro 附近的线缆要固定好

## 柔性 PCB 替代方案

如果你坚持想要 PCB 方案，有以下几种选择：

### 方案 1：使用刚性 PCB

**优点：** 设计简单，成本较低
**缺点：** 不适合 Dactyl 的 3D 曲面

一些社区成员为 Dactyl Manuform 设计了刚性 PCB：
- [Dactyl Manuform PCB by Carbonfet](https://github.com/carbonfet/dactyl-manuform-mini-keyboard)
- 需要修改外壳设计以适应平面 PCB

### 方案 2：柔性 PCB（FPC）

**优点：** 可弯曲，适合 3D 曲面
**缺点：** 设计复杂，制造成本高（通常 ¥300-500/张）

#### 设计柔性 PCB 的步骤：

1. **测量键位坐标：**
   - 使用打印的外壳测量每个键的精确位置
   - 记录 X, Y, Z 坐标和旋转角度

2. **设计 PCB：**
   - 使用 KiCad 或 Eagle
   - 为每个键位放置热插拔座焊盘
   - 布线形成矩阵

3. **分段设计：**
   - 将键盘分为几个区域（主区、拇指区）
   - 使用柔性连接器连接各区域

4. **制造：**
   - 订购柔性 PCB（推荐 JLCPCB、PCBWay）
   - 通常需要 5-10 张起订，单价较高

5. **组装和测试**

#### 柔性 PCB 设计示例：

```
主键区（3x5）：
┌─────────────────────────┐
│  O   O   O   O   O      │  O = 键位
│  O   O   O   O   O      │
│  O   O   O   O   O      │  柔性区域可弯曲
└──────────┬──────────────┘
           │ 柔性连接
           │
    ┌──────┴────┐
    │ 拇指区    │
    │  O    O   │
    └───────────┘
```

#### 成本估算：

- **设计时间：** 20-40 小时（如果熟悉 PCB 设计）
- **制造成本：** ¥300-500（5 张起订）
- **元件成本：** 与手工布线相同
- **总成本：** 约 ¥600-900（首次）

### 方案 3：半柔性方案

使用刚性 PCB + 柔性排线的混合方案：

1. **主区 PCB：** 每个键位区域使用小的刚性 PCB
2. **柔性排线：** 使用 FFC（Flat Flexible Cable）连接各个小 PCB
3. **背板：** 所有小 PCB 贴在 3D 打印的背板上

**优点：**
- 比纯柔性 PCB 便宜
- 可以购买现成的 FFC
- 设计相对简单

**缺点：**
- 仍需要设计多个小 PCB
- 组装复杂度高

### 方案 4：使用现成的分体式键盘 PCB

修改外壳以适应现成的 PCB：

1. **选择 PCB：**
   - Corne（3x6）
   - Lily58（4x6）
   - Sofle（5x6）

2. **修改外壳：**
   - 调整键位坐标以匹配 PCB
   - 可能需要放弃 3D 曲面，改为平面或轻微弯曲

3. **组装：**
   - 使用现成的 PCB
   - 3D 打印修改后的外壳

**优点：**
- PCB 成熟，无需设计
- 成本低（约 ¥50-100/PCB）

**缺点：**
- 失去 Dactyl 的完全定制特性
- 外壳修改工作量大

## 推荐方案对比

| 方案 | 成本 | 难度 | 时间 | 定制性 | 推荐指数 |
|------|------|------|------|--------|----------|
| **手工布线** | ¥200-300 | 中 | 20-40h | ★★★★★ | ★★★★★ |
| **柔性 PCB** | ¥600-900 | 高 | 40-60h | ★★★★☆ | ★★☆☆☆ |
| **半柔性** | ¥400-600 | 高 | 30-50h | ★★★☆☆ | ★★★☆☆ |
| **现成 PCB** | ¥300-400 | 低 | 10-20h | ★★☆☆☆ | ★★★★☆ |

### 建议：

1. **首次构建：** 手工布线
   - 成本低，灵活性高
   - 可以边学边做
   - 出问题容易调试

2. **追求整洁：** 现成 PCB + 修改外壳
   - 使用 Corne 或 Lily58 PCB
   - 修改外壳以适应平面布局

3. **有 PCB 设计经验：** 柔性 PCB
   - 完全定制
   - 需要投入大量时间和成本

## 结论

### 对于大多数用户：

**推荐使用手工布线方案。**

原因：
1. 本项目专为手工布线优化
2. 热插拔座设计已考虑手工布线的走线空间
3. 使用排针的可拆卸设计非常适合调试和维护
4. 成本低，灵活性高

### 如果你确实需要 PCB：

1. **考虑现成的分体式键盘 PCB** 并修改外壳
2. 或**设计自己的柔性 PCB**（需要投入时间和成本）

### 手工布线的改进建议：

1. **使用更粗的线缆：** README 提到细线容易断，建议 28 AWG
2. **多用排针：** 所有连接都使用排针，方便维护
3. **绝缘和固定：** 使用热缩管和热熔胶
4. **预留备用连接点：** 在关键位置预留额外的焊盘
